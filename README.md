# 壽司郎香港價錢計算器

一個自包含嘅 HTML 檔，幫你喺食壽司郎嗰陣即時計碟數同結算金額。打開即用，唔使裝任何嘢。

**🌐 線上版：** https://ed-lam-11111010010.github.io/sushiro-counter/

---

## 功能

- **四色碟計數** — 紅碟 $12 / 銀碟 $17 / 金碟 $22 / 黑碟 $27，每行 `− 數量 +` stepper，即時顯示每色小計
- **其他堂食單品** — 55 款預設項目分 4 類（麵類・湯類 / 副餐類 / 甜品 / 飲料），撳「加入其他項目」開小分頁揀，每項可調數量、可刪除
- **加成 / 折扣** — 支援 `+` 同 `−`、`$` 同 `%`，可加多行、可刪除（例如 −$20 優惠券、+10% 附加費）
- **即時結算** — sticky 結算欄顯示 小計 → 加成調整 → 加成後小計 → 服務費 10% → 總計
- **自動儲存** — localStorage 記住你嘅選擇，refresh 唔會清空；附「重置」掣一鍵清零
- **離線可用** — 純靜態檔，雙擊即開，唔使上網
- **手機優先** — responsive 設計，手機餐廳現場用都得
- **繁體中文（HK）** 介面

## 點計

計算順序（HK 餐廳慣例，折扣喺服務費之前計）：

```
小計       = Σ(碟數 × 碟價) + Σ(其他項目 數量 × 價錢)
加成調整   = Σ 所有加成/折扣行（$ 直接加減；% = 小計 × 百分比）
加成後小計 = max(0, 小計 + 加成調整)        ← 負數會 clamp 做 0
服務費     = 加成後小計 × 10%
總計       = 加成後小計 + 服務費
```

### 例子

紅碟 ×2、黑碟 ×1、茶碗蒸 ×1、豚骨拉麵 ×1，再用 −$20 優惠券：

| 項目 | 金額 |
|------|------|
| 紅碟 ×2 | $24 |
| 黑碟 ×1 | $27 |
| 茶碗蒸 ×1 | $19 |
| 豚骨拉麵 ×1 | $32 |
| **小計** | **$102** |
| 優惠券 −$20 | −$20 |
| **加成後小計** | **$82** |
| 服務費 10% | $8.20 |
| **總計** | **$90.20** |

## 自訂

打開 `index.html`，改檔案頂部嘅 config：

```js
const PLATES = [
  { id:'red',    label:'紅碟', price:12, color:'#e53935', ... },
  { id:'silver', label:'銀碟', price:17, color:'#757575', ... },
  // 改 price 就得；物價變時改呢度
];

const OTHER_ITEMS = {
  '麵類・湯類': [ { id:'steamed_clams', name:'出汁蒸蜆', price:18 }, ... ],
  '副餐類':     [ ... ],
  '甜品':       [ ... ],
  '飲料':       [ ... ],
  // 加減項目、改價錢都喺呢度
};
```

改完 save 就得，唔使重新 build 任何嘢。

## 自我測試

喺瀏覽器 console 跑：

```js
window.__selftest()
```

會用固定輸入測試小計、加成、服務費、總計嘅計算是否正確，並 log ✅/❌ 結果。
